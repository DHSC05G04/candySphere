## Uso de DB

Para usar DB, utilizaremos o modulo DOTENV do node.

Criar um arquivo `.env` na raiz do seu projeto com o conteúdo abaixo.

```txt
PORT=3000
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=suaSenhaAqui
```
